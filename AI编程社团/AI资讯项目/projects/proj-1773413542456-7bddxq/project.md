# AI资讯项目

## 项目目标

根据AI News Skill自动聚合每日AI资讯，推送到飞书机器人，实现每日AI新闻自动追踪和推送。

## 核心功能

1. **多源聚合**: 从15+个权威AI资讯源获取最新内容
2. **智能筛选**: 多维度评分算法筛选最值得关注的新闻
3. **自动分类**: AI驱动分类（技术突破/产品发布/研究论文/行业动态/政策法规/工具教程）
4. **去重过滤**: URL+相似度双重去重
5. **飞书推送**: 自动推送到飞书群机器人
6. **定时执行**: 每日自动更新并推送

## 技术栈

- Python 3.x
- feedparser (RSS采集)
- SQLite (本地存储)
- schedule (定时任务)
- PyYAML (配置管理)
